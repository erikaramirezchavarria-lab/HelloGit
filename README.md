# Proyecto HelloGit – Prácticas de Git y GitHub  

Este repositorio forma parte de las prácticas realizadas en la asignatura de **Control de Versiones**, donde aplico los comandos y flujos de trabajo más importantes de **Git** y **GitHub**.

---

##  Descripción general

Este proyecto es un repositorio de práctica que incluye:

- Creación de repositorios con Git  
- Manejo de ramas  
- Fusiones (merge)  
- Manejo de versiones  
- Uso de Markdown  
- Trabajo con repositorio remoto  
- Fork y Pull Request hacia el repositorio de la maestra
